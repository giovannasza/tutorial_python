CONTEXTUALIZAÇÃO
APIs são usadas para integrar sistemas de diversas maneiras.
APIs WEB ganham cada vez mais popularidade por usar o protocolo HTTP
e padronizar a comunicação em um só meio.
Antes cada API possuia sua porta e padrão de comunicação, o que inviabilizava
o processo de integração entre sistemas, com o protocolo HTTP todo o trafego
passa a ser na WEB nas portas 80 ou 443 comumente vigiadas por firewalls, trazendo
maior confiabilidade e segurança para usuários e desenvolvedores.
